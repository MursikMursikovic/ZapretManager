> **Графический интерфейс (GUI) для управления `zapret-discord-youtube`**  
> Удобная оболочка для запуска, остановки и настройки обхода блокировок Discord, YouTube и Telegram.

## 📌 О проекте

**ZapretManager** — это простая и интуитивно понятная программа с графическим интерфейсом для управления утилитой [zapret-discord-youtube](https://github.com/Flowseal/zapret-discord-youtube). Больше никаких BAT-файлов и командной строки — всё управление в одном окне.

### ✨ Возможности

- 🎯 **Выбор стратегии** — запуск любой из 20+ стратегий обхода из выпадающего списка.
- ▶️ **Управление** — кнопки "Запустить", "Остановить" и "Перезапустить".
- 📊 **Статус в реальном времени** — отображение состояния процессов и их PID.
- 🔧 **Управление службами** — установка и удаление автозапуска Windows.
- 🎮 **Game Filter** — переключение режима обхода для игр.
- 🌐 **IPSet Filter** — гибкое управление фильтрацией IP-адресов.
- 🗂️ **Обновление списков** — обновление IP-адресов и доменов одним кликом.
- 📝 **Обновление hosts** — автоматическая правка файла hosts для веб-версий сервисов.
- 🔍 **Диагностика** — проверка распространённых проблем.
- 📋 **Лог операций** — все действия записываются в удобный лог с возможностью сохранения.
- 🔐 **Права администратора** — автоматический запрос прав при запуске.

## 🛠️ Установка и запуск

### 📦 Готовый EXE-файл (для обычных пользователей)

1. Скачайте последний релиз со страницы [Releases](https://github.com/MursikMursikovic/ZapretManager/releases).
2. Распакуйте архив в папку, где находится `zapret-discord-youtube` (или укажите путь в программе).
3. Запустите `ZapretGUI.exe` **от имени администратора** (программа запросит автоматически).
4. Выберите стратегию и нажмите "Запустить".

### 🐍 Запуск из исходного кода (для разработчиков)

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/ваш-проект/zapret-gui-controller.git
cd zapret-gui-controller

# 2. Установите зависимости
pip install psutil

# 3. Запустите GUI
python zapret_gui.py
